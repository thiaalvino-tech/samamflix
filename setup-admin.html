BRASFLIX • Setup do primeiro admin

Esta tela resolve o problema do primeiro administrador sem usar terminal. O fluxo é seguro: você precisa estar logado e informar um segredo de bootstrap salvo na Vercel como SETUP_ADMIN_SECRET. A promoção só acontece se ainda não existir nenhum admin no Firestore.

Checklist

Você precisa ter na Vercel:
FIREBASE_ADMIN_PROJECT_ID FIREBASE_ADMIN_CLIENT_EMAIL FIREBASE_ADMIN_PRIVATE_KEY SETUP_ADMIN_SECRET
Quando der certo:
seu documento em usuarios/{uid} passa a ter role: "admin" e as custom claims também são sincronizadas.
  1. Crie uma senha forte para o segredo.
  2. Salve a variável na Vercel e faça redeploy.
  3. Abra esta tela já autenticado.
  4. Promova apenas o primeiro admin por aqui.
  5. Depois use o admin-setter para gerenciar os demais.
